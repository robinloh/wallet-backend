## Prerequisities
1. [Golang](https://go.dev/doc/install)
2. [Docker Desktop](https://docs.docker.com/engine)


## How to run

In the terminal, ensure that the current directory is  `wallet-backend`.

Then, run the following command.

```docker compose up```
